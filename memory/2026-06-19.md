# 2026-06-19 (Пт)

## Avatar Service — Gemini API Key повреждён на проде

**Что произошло:**
- Пользователь сообщил, что на проде не генерируются картинки
- В логах: Gemini API возвращает 401 (UNAUTHENTICATED)
- Причина: в `/opt/avatar-service-prod/.env` ключ `GEMINI_API_KEY` оказался повреждён — вместо нормального ключа (39 символов) лежала бинарная каша (~17 символов с нечитаемыми байтами)
- Ключ восстановлен из тестового `.env`, бот перезапущен

**Корень:** неизвестен — возможно, файл повредился при одной из предыдущих раскаток или записи.

**Выводы:**
1. `.env` продублирован на тесте — хорошо, ключ не потерялся
2. Нужен бэкап `.env` сам по себе + мониторинг здоровья ключа при старте
3. При раскатке не трогать `.env` — но проблема могла возникнуть не из-за раскатки, а из-за случайной перезаписи

**Деплой скрипты:**
- Добавлены в git в `avatar-service/deploy/` по просьбе пользователя
