### Дублирование процессов бота
- Проблема: при перезапусках оставался старый процесс → два бота поллинжили Telegram → дубли ответов
- **Решение:** добавлен PID lock `/tmp/imgy-bot.pid` в `bot-runner.js`
  - При запуске проверяет, жив ли процесс из lock-файла
  - Если жив — `process.exit(1)` с ошибкой
  - Если мёртв — удаляет мусор и пишет свой PID
  - Автоочистка на `exit`, `SIGINT`, `SIGTERM`, `SIGHUP`
- Также обнаружен systemd-сервис `avatar-bot.service` (автостарт, `Restart=always`)
- **Договорённость:** рестарт после правок только через `systemctl restart avatar-bot`
