## Как использовать

Эти промпты — база. Для генерации можно:
- **Сокращать** — убрать лишние детали если модель не справляется
- **Усиливать** — добавить `cinematic lighting`, `octane render`, `photorealistic` для реализма
- **Комбинировать** — например, смешать промпт №26 (Hive City) с №16 (Guard)

Файл можно редактировать, добавлять, перетасовывать. 🌀
